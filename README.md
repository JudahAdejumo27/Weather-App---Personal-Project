# Weather-App---Personal-Project
This is a personal project I worked on, it's a simple Weather App made using Javascript for the programming language in Visual Code Studio. It showcases the current weather of any city, province or state using a website called OpenWeather, as the source for tracking down the weather data. The Link for said source can be displayed below.
https://openweathermap.org/

The units were in degrees Celsius with the inclusion of the humidity, wind speed and current timezone in cities like Paris or London. You may download the files and take a look at how it was made by downloading the files in the main branch.
